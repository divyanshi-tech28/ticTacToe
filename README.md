# Tic-Toc-Toe-Game-against-AI
Tic Tac Toe game built with vanilla JavaScript featuring an AI opponent. The AI defends against player winning moves and attempts to win when possible. Includes hover effects, winner highlighting, tie detection, restart functionality, and game state persistence with localStorage.
# Tic Tac Toe Against AI

A classic Tic Tac Toe game built with vanilla JavaScript where you play against an AI opponent.

## Live Demo
https://bekamanjgaladze3-dev.github.io/Tic-Toc-Toe-Game-against-AI/

## Features
- AI opponent that defends against your winning moves and attempts to win
- Hover effects on empty cells
- Winner cells highlighted when game ends
- Tie detection
- Restart functionality
- Game state saved with localStorage — refresh the page and continue your game

## Built With
- HTML
- CSS
- Vanilla JavaScript

## How To Play
1. You play as **X**, AI plays as **O**
2. Click any cell to make your move
3. AI responds automatically
4. First to get 3 in a row wins
5. Click **Restart** to start a new game

## Getting Started
```bash
git clone https://github.com/bekamanjgaladze3-dev/Tic-Toc-Toe-Game-against-AI.git
cd tic-tac-toe
open index.html
```
